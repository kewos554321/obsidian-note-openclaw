#### 📚 Today's Concept: How to read an earnings report

What it is: An earnings report is a quarterly or annual financial statement public companies release, showing revenue, expenses, net income, and earnings per share (EPS) for the past period. It also includes forward-looking guidance on expected future performance.

Why it matters: You compare reported numbers to analysts' consensus estimates to gauge whether the stock beat or missed expectations, which often drives immediate price moves. For a software engineer, think of it like a build passing or failing tests—the market reacts to the delta between expected and actual results.

Example: Company X reports $1.20 EPS on $5B revenue, while analysts expected $1.10 EPS on $4.8B revenue. The stock jumps 8% after hours because both metrics beat estimates. If revenue had been $4.6B (a miss), the stock might drop 5% despite an EPS beat.

Rule of thumb: Focus on revenue growth and forward guidance, not just EPS—companies can manipulate EPS with buybacks or one-time gains. A warning sign is when revenue grows slower than expenses, indicating shrinking margins.

## 131. Palindrome Partitioning [Backtracking]

**Problem Type:** Backtracking / Subset Generation

**Key Insight:** At each position, try all possible substrings starting from that position; if the substring is a palindrome, recurse on the remaining string.

**Approach:**
1. Define a recursive backtracking function `dfs(start, path)` where `start` is the current index in `s`
2. Base case: if `start == len(s)`, add a copy of `path` to result
3. For each `end` from `start` to `len(s)-1`:
   - Check if `s[start:end+1]` is a palindrome
   - If yes, add it to `path`, recurse with `dfs(end+1, path)`, then backtrack
4. Use a helper `is_palindrome(s, l, r)` with two-pointer check for O(1) substring check

**Python3 Solution:**
```python
class Solution:
    def partition(self, s: str) -> List[List[str]]:
        def is_pal(l: int, r: int) -> bool:
            while l < r:
                if s[l] != s[r]:
                    return False
                l += 1
                r -= 1
            return True
        
        res = []
        path = []
        
        def dfs(start: int) -> None:
            if start == len(s):
                res.append(path[:])
                return
            for end in range(start, len(s)):
                if is_pal(start, end):
                    path.append(s[start:end+1])
                    dfs(end + 1)
                    path.pop()
        
        dfs(0)
        return res
```

**Complexity:** Time O(n * 2^n) | Space O(n) for recursion stack + path

**Blind 100 Note:** Classic backtracking pattern for generating all valid partitions. Tests ability to combine substring generation with constraint checking. Similar problems: *Combination Sum*, *Subsets*, *Word Break II*.

**Contest Tips:**
- Use two-pointer palindrome check (not `s[::-1]`) to avoid O(n) string slicing overhead
- Pass indices instead of substrings to `is_pal` for efficiency
- Remember `path[:]` copy when adding to results
- Edge case: single character strings (always palindrome) → returns `[[s]]`

#### 📷 Today's Concept: Video — 180-Degree Shutter Rule for Natural Motion Blur

**What it is:** The 180-degree shutter rule sets your shutter speed to double your frame rate (e.g., 1/50s for 24fps). This creates natural motion blur that mimics how our eyes perceive movement.

**Why it matters:** Correct blur makes video look smooth and cinematic; too fast looks stroboscopic, too slow looks smeary. It’s the foundation of professional motion.

**How to apply it:**
1. Set your frame rate (e.g., 24fps for cinematic look).
2. Set shutter speed to double: 1/50s for 24fps, 1/100s for 50fps.
3. Adjust aperture and ISO to correct exposure (use ND filter in bright light).
4. Lock shutter speed—never change it mid-shot unless going for an effect.

**Sony A7C tip:** Use **S&Q mode** (Slow & Quick) for 24fps recording at 1/50s. Set **ISO Auto Min. SS** to 1/50s in movie mode to avoid accidental shutter changes.

**Common mistake:** Shooting at 1/125s or faster in bright daylight without an ND filter, creating jarring, choppy motion. **Fix:** Buy a variable ND filter (e.g., K&F Concept) to maintain 1/50s in any light.

#### 📚 Today's Concept: API Versioning Strategies

**What it is:** API versioning is the practice of managing changes to an API without breaking existing clients. It allows you to introduce new features or modify behavior while keeping older integrations functional.

**When to use it:** Use it when your API is consumed by external clients (e.g., mobile apps, third-party services) that you cannot update simultaneously. *Real-world scenario:* A payment gateway adds a new required field for fraud detection; existing checkout apps must continue working without that field.

**Example (URL path versioning):**
```
GET /api/v1/users/123   # returns { "name": "Alice" }
GET /api/v2/users/123   # returns { "name": "Alice", "email": "alice@example.com" }
```

**Gotcha:** Assuming versioning means you never have to deprecate old versions. Without sunsetting old versions, you accumulate technical debt and maintenance burden—always set a clear deprecation timeline (e.g., “v1 will be removed 6 months after v2 release”).

3. **用 AI 工具快速剪輯**：試試「剪映」自動生成字幕、刪除靜音片段，省時 50%。  
4. **控制節奏**：每 10-15 秒換畫面，搭配背景音樂（如 Epidemic Sound 免費版）。  
5. **結尾留鉤子**：預告下集內容（如「明天開箱神秘包裹」）。

**新手常犯的錯：**  
**拍太多無意義畫面**（如走路 3 分鐘）。→ 只保留「動作」與「反應」，其餘刪除。

**延伸 idea：**  
**「用 Sony A7C 拍一支 3 分鐘的『房間改造 Vlog』」**：從規劃、採買到完工，穿插 AI 生成縮時，適合攝影與生活頻道。
